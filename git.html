Les commandes de bases

  • Pour vérifier les options de configuration git config --list
  • Pour initialiser un dépôt git init
  • Ignore des fichiers à indexer gitignore
  • Ajout des fichiers à indexer git add $.c
  • Commit les fichiers indexés git commit -m 'Message d'information'

Les fichiers à ignorer

  • .classpath
  • .project
  • .wtpmodules

Git les principes

Pour fonctionner, Git a besoin de 3 choses :
  • un répertoire de travail (le projet dans le workspace d'Eclipse par exemple),
  • un dépôt,
  • un index.
